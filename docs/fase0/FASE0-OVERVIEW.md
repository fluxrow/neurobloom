# Fase 0 — Mapeamento do Perfil (em construção)

> A Fase 0 é o DNA NeuroBloom. É onde tudo começa e de onde tudo deriva.
> Sem ela completa, o app é genérico. Com ela, é único para cada criança.

---

## Objetivo da Fase 0

Coletar informações suficientemente profundas para:

1. Montar o perfil real da criança (diagnósticos, comorbidades, pontos fortes, perfil sensorial, interesses)
2. Calibrar todas as camadas seguintes (1–4) com precisão
3. Funcionar como primeira "consulta estruturada" para famílias sem acesso a profissionais
4. Construir um banco de dados longitudinal sobre neurodivergência no Brasil

---

## Princípios do Onboarding

- **Funciona sem laudo** — guia os pais a observar e descrever com profundidade clínica
- **Absorve laudos quando existem** — extrai o máximo de informação técnica disponível
- **Progressivo** — começa com o essencial, aprofunda ao longo do tempo
- **Empático** — linguagem acessível, sem jargão, sem julgamento
- **Baseado em estudos públicos** — cada pergunta fundamentada em literatura científica
