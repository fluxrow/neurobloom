# 🌱 NeuroBloom — Master Document

> Plataforma mobile-first para potencializar o desenvolvimento de crianças e adultos no espectro autista, com personalização profunda por perfil de comorbidades.

---

## Visão do Produto

**Tagline:** Potencializar cada mente. Honrar cada caminho.

**Problema central:** A maioria das famílias com filhos no espectro não tem acesso a terapias especializadas. O NeuroBloom é a ferramenta estruturada para quem não tem nada — e o complemento inteligente para quem tem.

**North Star Metric:** Progressos reais conquistados pela criança, celebrados pelos pais, registrados pelo app — semana a semana.

---

## Princípios Inegociáveis

1. **Baseado em evidências** — medicina, estudos e terapias reais. Sem achismo.
2. **Não substitui profissionais** — somos o que acontece nas outras 23h do dia.
3. **Personalização real** — não "tem TEA", mas qual combinação exata de comorbidades, nível de suporte, pontos fortes, perfil sensorial.
4. **Acessível** — funciona sem laudo. Para famílias sem acesso a terapeutas.
5. **Multimodal** — voz, texto, imagem, toque, animação, vibração. Cada criança tem um canal preferencial.
6. **Dados com propósito** — cada família que usa está construindo o maior banco de dados sobre neurodivergência do Brasil.

## Arquitetura de Camadas

| Camada | Nome | Foco Principal |
|--------|------|----------------|
| **0** | Mapeamento do Perfil | Onboarding profundo — DNA NeuroBloom da criança |
| **1** | Fundação e Regulação Sensorial | Sono, alimentação, rotinas, integração sensorial |
| **2** | Habilidades Funcionais | Comunicação, atenção, imitação, AVDs |
| **3** | Potencial e Talentos | Cognitivo, criativo, acadêmico, altas habilidades |
| **4** | Social e Autonomia | Habilidades sociais, teoria da mente, independência |

> A Fase 0 é a mais crítica. Sem ela completa, as demais camadas são genéricas. Com ela, o app se torna único para cada criança.

## Base Metodológica

### Evidência Nível A — Core
- **ABA** (Applied Behavior Analysis) — micro-passos, reforço positivo, coleta de dados contínua
- **ESDM** (Early Start Denver Model) — relacional, baseado em jogo, pais como co-terapeutas

### Complementar Validado
- **Floortime / DIR + RDI** — círculos de comunicação, conexão emocional como fundação
- **Terapia Ocupacional + Integração Sensorial** — dieta sensorial personalizada por perfil
- **Interoceptividade (Mahler/IC)** — o 8º sentido: regulação emocional via consciência corporal

### Abordagens Emergentes (monitoradas, não prescritivas)
- Eixo intestino-cérebro / microbioma
- Neurofeedback e biofeedback via câmera / wearable
- VR/AR para habilidades sociais
- IA adaptativa e digital biomarkers
- TMS/tDCS — informativo
- Modulação endocanabinoide — informativo

## Mapa de Comorbidades

| Comorbidade | Prevalência no TEA | Impacto no app |
|-------------|-------------------|----------------|
| Processamento sensorial (SPD) | ~90% | Design do app, dieta sensorial |
| TDAH | 50–70% | Micro-sessões, foco, impulsividade |
| Distúrbio de sono | 50–80% | Módulo de rotinas, higiene do sono |
| Epilepsia | ~25% | Alertas, sem flashes, adaptações visuais |
| Ansiedade | ~28% | Regulação emocional, interoceptividade |
| Altas Habilidades (AH/SD) | ~17% | Desafios acima da média, anti-mascaramento |
| TOC | ~17% | Flexibilização, rotinas graduais |
| TDE (dislexia/disgrafia) | 5–10% | Multimodalidade, áudio-primeiro |

**Caso real interno:** TEA Nível 1 + TDAH + TDE + Altas Habilidades. Perfil que guia todas as decisões de design e UX.
