# 🌱 NeuroBloom

> Plataforma mobile-first para potencializar o desenvolvimento de crianças e adultos no espectro autista, com personalização profunda por perfil de comorbidades.

**Tagline:** Potencializar cada mente. Honrar cada caminho.

## O Problema

A maioria das famílias com filhos no espectro autista no Brasil não tem acesso a terapias especializadas. O NeuroBloom é a ferramenta estruturada para quem não tem nada, e o complemento inteligente para quem tem.

## Arquitetura de Camadas

| Camada | Nome | Foco |
|--------|------|------|
| 0 | Mapeamento do Perfil | Onboarding profundo — DNA NeuroBloom |
| 1 | Fundação e Regulação Sensorial | Sono, alimentação, rotinas |
| 2 | Habilidades Funcionais | Comunicação, atenção, AVDs |
| 3 | Potencial e Talentos | Cognitivo, criativo, altas habilidades |
| 4 | Social e Autonomia | Habilidades sociais, independência |

## Base Metodológica

- ABA — Applied Behavior Analysis (evidência nível A)
- ESDM — Early Start Denver Model (evidência nível A)
- Interoceptividade (Mahler/IC Curriculum)
- Integração Sensorial (TO)
- Abordagens emergentes: microbioma, neurofeedback, IA adaptativa
